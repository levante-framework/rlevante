# Get raw trial data

Get raw trial data

## Usage

``` r
get_trials_raw(dataset_spec)
```

## Arguments

- dataset_spec:

  List of dataset names and versions to retrieve.

## Examples

``` r
if (FALSE) { # \dontrun{
dataset_spec <- list(list(name = "levante-example-dataset:bm7r", version = "current"))
trials_raw <- get_trials_raw(dataset_spec)
} # }
```
