# score

score

## Usage

``` r
score(task, dataset, trials, runs, scoring_table, registry_table)
```

## Arguments

- task:

  task

- dataset:

  dataset

- trials:

  trial data from one task and one dataset

- runs:

  run data from one task and one dataset

- scoring_table:

  table returned by get_scoring_table

- registry_table:

  table returned by get_registry_table
