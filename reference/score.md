# score

score

## Usage

``` r
score(task, dataset, trials, runs, scoring_table, registry_dir)
```

## Arguments

- task:

  task

- dataset:

  dataset

- trials:

  trial data from one task and one dataset

- runs:

  run data from one task and one dataset

- scoring_table:

  tibble returned by fetch_scoring_table()

- registry_dir:

  redivis directory returned by fetch_registry_dir()
