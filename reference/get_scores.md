# Get score data

Get score data

## Usage

``` r
get_scores(dataset_spec)
```

## Arguments

- dataset_spec:

  List of dataset names and versions to retrieve.

## Examples

``` r
if (FALSE) { # \dontrun{
dataset_spec <- list(list(name = "levante-example-dataset:bm7r", version = "current"))
scores <- get_scores(dataset_spec)
} # }
```
