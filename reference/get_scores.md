# Get scores

Get scores

## Usage

``` r
get_scores(data_source, version = "current")
```

## Arguments

- data_source:

  Name of Redivis dataset

- version:

  Version of Redivis dataset

``` r
if (FALSE) { # \dontrun{
scores <- get_scores(dataset = "levante-data-example:bm7r", version = "current")
} # }
```
