# Get surveys

Get surveys

## Usage

``` r
get_surveys(data_source, version = "current")
```

## Arguments

- data_source:

  Name of Redivis dataset

- version:

  Version of Redivis dataset

``` r
if (FALSE) { # \dontrun{
```
