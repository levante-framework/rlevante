# fetch file from model registry

fetch file from model registry

## Usage

``` r
get_registry_file(mod_filename, registry_dir)
```

## Arguments

- mod_filename:

  Filename (string) of model record to fetch.

- registry_dir:

  Model registry directory as returned by
  [`fetch_registry_dir()`](https://levante-framework.github.io/levantemodels/reference/fetch_registry_dir.md).
